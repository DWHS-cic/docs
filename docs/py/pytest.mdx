# Python是什麼？
    Python 是一種高階程式語言，以簡單易讀、易學習和靈活性著稱。

## Python特點
    1. 語言特性
        - 簡潔易讀
            - Python 的語法設計簡單、接近自然語言，因此適合初學者快速上手。
        - 跨平台
            - Python 可以在 Windows、Mac、Linux 等多種作業系統上執行。
        - 動態型別
            - 變數不需要明確宣告型別，程式在執行時自動處理型別。
	    - 龐大的標準庫
            - 內建多種模組，涵蓋文件處理、網路協定、數學運算等，降低外部依賴。

    2. 核心用途
        - 網站開發
            - 使用框架如 Django、Flask 可快速構建後端服務。

        - 數據科學與機器學習
	        - 數據處理：使用 Pandas、NumPy 等進行數據分析。
	        - 視覺化：Matplotlib、Seaborn 用於製作圖表。
	        - 機器學習：使用 Scikit-learn、TensorFlow 或 PyTorch 進行模型訓練。

        - 自動化腳本
            - 自動化日常任務，例如批量處理檔案、網路爬蟲等。

        - 嵌入式與物聯網
            - Python 可用於 Raspberry Pi 等硬體的快速開發。

        - 遊戲開發
            - 利用 Pygame 創建 2D 遊戲。

        - 科學計算
            - 適合模擬、統計分析和其他科學領域應用。
