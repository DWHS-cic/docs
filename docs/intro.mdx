# 開發環境

## Dev C++
一款免費的 C/C++ 集成開發環境，它是一個 Windows 環境下的 IDE，包含了 MinGW 編譯器
:::warning
Dev-C++ 已經停止開發，最後一個版本為 5.11<br />不支援 macOS 和 Linux
:::

### 安裝
一. 下載 [Dev C++](https://sourceforge.net/projects/orwelldevcpp/)
    1. 開啟瀏覽器，前往 [Dev C++ 官網](https://sourceforge.net/projects/orwelldevcpp/)
    2. 點擊 `Download` 下載 `.exe` 檔案

二. 安裝 Dev C++
    1. 開啟下載的 `.exe` 檔案
    2. 按照安裝指示完成安裝

## Visual Studio Code
一款由微軟開發的跨平臺的程式碼編輯器，支援 Windows、macOS 和 Linux

### 安裝
<Tabs groupId="tutorial-extras">
  <TabItem value="MacOS" label="MacOS">
    ㄧ. 下載 [Visual Studio Code](https://code.visualstudio.com)
        1. 開啟瀏覽器，前往 [Visual Studio Code 官網](https://code.visualstudio.com)
        2. 點擊 `Download for Mac` 下載 `.zip` 檔案

    二. 安裝 Visual Studio Code
      1. 開啟下載的 `.zip` 檔案
      2. 將 `Visual Studio Code.app` 拖曳至 `Applications` 資料夾
  </TabItem>
  <TabItem value="Windows" label="Windows">
    ㄧ. 下載 [Visual Studio Code](https://code.visualstudio.com)
        1. 開啟瀏覽器，前往 [Visual Studio Code 官網](https://code.visualstudio.com)
        2. 點擊 `Download for Windows` 下載 `.exe` 檔案

    二. 安裝 Visual Studio Code
        1. 開啟下載的 `.exe` 檔案
        2. 按照安裝指示完成安裝
  </TabItem>
</Tabs>

## 線上編譯器 onlinegdb
一款線上的 C/C++ 編譯器，支援多種語言

### 使用
一. 前往 [onlinegdb](https://www.onlinegdb.com)
    1. 開啟瀏覽器，前往 [onlinegdb 官網](https://www.onlinegdb.com)
    2. 點擊 `Start Coding` 開始編寫程式
