# 安裝環境

## [Dev-C++](https://www.bloodshed.net)
一款免費的 C/C++ 集成開發環墮，它是一個 Windows 環境下的 IDE，包含了 MinGW 編譯器
:::warning
Dev-C++ 已經停止開發，最後一個版本是 5.11（不支援 macOS 和 Linux）
:::
